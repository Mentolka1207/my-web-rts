# 🛰️ Minimal Web RTS: Neo-Drone Swarm & Patrolling AI

Минималистичный трёхмерный прототип стратегии в реальном времени (RTS) на **Three.js** (WebGL), демонстрирующий продвинутые алгоритмы поиска пути, группового поведения и искусственного интеллекта юнитов — с полноценной поддержкой мобильных устройств.

Проект разработан без внешних тяжёлых зависимостей и работает прямо в браузере.

---

## 🚀 Как запустить

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Mentolka1207/my-web-rts.git
   ```
2. Откройте файл `index.html` в любом современном браузере (десктоп или мобильный).

## 🛠️ Технологический стек и архитектура

В проекте реализован ряд фундаментальных систем, использующихся в масштабных коммерческих RTS (таких как Supreme Commander):

### Поиск пути на основе Flow Fields (Поля потоков)

Вместо индивидуального расчёта алгоритма A* для каждого юнита (что привело бы к падению производительности при сотнях объектов), движение реализовано через глобальные векторные поля:

- **Cost Field** (карта стоимостей) — сетка проходимости с препятствиями.
- **Integration Field** (карта интеграции) — расчёт волны расстояний до цели методом BFS (алгоритм Дейкстры).
- **Flow Field** (поле направлений) — генерация векторов движения в сторону наименьшего сопротивления. Юниты просто «плывут» по векторам, огибая любые препятствия.

### Динамические групповые построения (Formations)

- Поддержка двух типов строя: **Шеренга** (Line) и **Клин** (Wedge).
- Автоматический разворот локальной матрицы смещения строя по вектору движения всей группы.
- Алгоритм плавного слияния векторов: при приближении к цели юниты переключаются со следования общему Flow Field на занятие своих точных персональных позиций в строю.

### Боевая система и ИИ на основе конечного автомата (FSM)

Юниты функционируют на базе расширяемой стейт-машины со следующими состояниями:

- **IDLE** (ожидание) — пассивное сканирование местности на наличие врагов.
- **MOVE** (движение) — следование к цели с возможностью отвлечься на угрозу.
- **CHASE** (преследование) — выход из строя и сближение с конкретной целью.
- **ATTACK** (атака) — остановка на дистанции огня, расчёт кулдауна оружия и стрельба лазерами.

### Ресурсы и производство юнитов

- Ресурс «Энергия» стартует со 100 единиц и растёт пассивно (+3 в секунду).
- Кнопка **«Построить дрона (40⚡)»** тратит энергию и производит нового союзного дрона у точки базы, автоматически находя ближайшую свободную от стен и юнитов клетку.

### Камера: панорама и зум

- Камера построена на модели «точка обзора + направление»: панорама (WASD / тач) и зум (колесо мыши / щипок) работают независимо, сохраняя исходный угол обзора при любом уровне приближения.

### Миникарта

- Компактная карта в углу экрана показывает стены, союзников (циан) и врагов (красный), а также рамку текущей области обзора камеры.
- Клик (десктоп) или тап (тач) по миникарте мгновенно переносит камеру в указанную точку.

## ✨ Дополнительные фичи

- **3D Billboarding**: полоски здоровья всегда развёрнуты лицом к камере и плавно сжимаются от края при получении урона.
- **Процедурные текстуры**: все sci-fi текстуры генерируются динамически через HTML5 Canvas при запуске игры, что решает проблему CORS-блокировок при локальном открытии файла.
- **Локальная физика**: двухуровневая система расталкивания (юнит-юнит и юнит-стена) предотвращает слипание объектов.

## 🎮 Управление (десктоп)

| Действие | Управление |
|---|---|
| Выделить одного юнита | Клик ЛКМ |
| Выделить группу рамкой | Зажать ЛКМ и тянуть |
| Отдать приказ на движение | Клик ПКМ (с учётом выбранного построения) |
| Зум камеры | Колесо мыши |
| Панорама камеры | W, A, S, D или стрелки |
| Заспавнить вражеского дрона под курсором | Клавиша E |
| Свернуть/развернуть эту панель | Кнопка «Скрыть / Управление» |

## 📱 Управление (мобильные устройства / тач)

| Действие | Управление |
|---|---|
| Выделить своего дрона | Тап по нему |
| Отдать приказ на движение | Тап по земле при активном выделении |
| Выделить группу рамкой | Тянуть одним пальцем |
| Зум камеры | Свести/развести два пальца (pinch) |
| Панорама камеры | Вести двумя пальцами по экрану |
| Заспавнить вражеского дрона (замена клавише E) | Кнопка «Спавн врага» в панели управления |
| Перейти камерой в точку карты | Тап по миникарте |
