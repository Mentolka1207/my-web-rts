# 🛰️ Minimal Web RTS: Flow Fields & FSM Combat Engine

Минималистичный трехмерный прототип стратегии в реальном времени (RTS) на **Three.js** (WebGL), демонстрирующий продвинутые алгоритмы поиска пути, группового поведения и искусственного интеллекта юнитов.

Проект разработан без внешних тяжелых зависимостей и работает прямо в браузере.

🛠️ Технологический стек и архитектура

В проекте реализован ряд фундаментальных систем, использующихся в масштабных коммерческих RTS (таких как Supreme Commander):

1. Поиск пути на основе Flow Fields (Поля потоков)Вместо индивидуального расчета алгоритма $A^*$ для каждого юнита (что привело бы к падению производительности при сотнях объектов), движение реализовано через глобальные векторные поля:
2. Cost Field (Карта стоимостей): Сетка проходимости с препятствиями.
3. Integration Field (Карта интеграции): Расчет волны расстояний до цели методом BFS (алгоритм Дейкстры).
4. Flow Field (Поле направлений): Генерация векторов движения в сторону наименьшего сопротивления. Юниты просто «плывут» по векторам, огибая любые препятствия.
5. Динамические групповые построения (Formations)
6. Поддержка двух типов строя: Шеренга (Line) и Клин (Wedge).
7. Автоматический разворот локальной матрицы смещения строя по вектору движения всей группы.
8. Алгоритм плавного слияния векторов: при приближении к цели юниты переключаются со следования общему Flow Field на занятие своих точных персональных позиций в строю.

 Боевая система и ИИ на основе Конечного автомата (FSM)Юниты функционируют на базе расширяемой стейт-машины со следующими состояниями:
1. IDLE (Ожидание) — пассивное сканирование местности на наличие врагов.
2. MOVE (Движение) — следование к цели с возможностью отвлечься на угрозу.
3. CHASE (Преследование) — выход из строя и сближение с конкретной целью.
4. ATTACK (Атака) — остановка на дистанции огня, расчет кулдауна оружия и стрельба лазерами.

 Дополнительные фичи: 
 1. 3D Billboarding: Полоски здоровья всегда развернуты лицом к камере и плавно сжимаются от края при получении урона.
 
2. Процедурные текстуры: Все Sci-Fi текстуры генерируются динамически через HTML5 Canvas при запуске игры, что решает проблему CORS-блокировок при локальном открытии файла.

3. Локальная физика: Двухуровневая система расталкивания (юнит-юнит и юнит-стена) предотвращает слипание объектов.

🎮 Управление через ЛКМ (Клик): Выделить одного юнита.

ЛКМ (Зажатие и перетаскивание): Групповое выделение рамкой.

ПКМ (Клик): Отдать приказ на движение в точку (с учетом выбранного построения).

Клавиша E: Создать вражеского дрона в точке под курсором мыши.

Клавиши W, A, S, D / Стрелочки: Свободное перемещение RTS-камеры по карте.
